# OpenLakes.io Website

This repository now contains only the OpenLakes.io marketing landing page (served at <https://openlakes.io>). All documentation content has moved to <https://github.com/OpenLakes/docs> and is published at <https://docs.openlakes.io>.

## Documentation

- Source repo: <https://github.com/OpenLakes/docs>
- Live site: <https://docs.openlakes.io>

Contribute to docs by opening PRs in the `OpenLakes/docs` project.

## Local Development (Landing Page)

The landing page is static HTML/CSS/JS. To preview locally:

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.
