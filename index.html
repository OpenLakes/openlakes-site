Why OpenLakes?

Apache 2.0 licensed reference architecture with the practical defaults needed to operate a reliable lakehouse.

Guided install process

Use the deployer script to generate manifests, review them in Git, and roll out the stack with the level of automation that fits your change controls.

Open formats by default

Everything ships in Iceberg, Parquet, and open APIs so you can swap engines, clouds, or storage without rewriting workloads.

Right-sized footprints

Profiles for lab clusters, PoCs, and multi-node production give you a clear bill of materials before you turn anything on.

Operational guardrails

Baseline security, monitoring, and backup tasks are included so you start with sensible defaults instead of a blank Terraform folder.
